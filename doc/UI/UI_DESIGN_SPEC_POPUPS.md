### 5.4 交互说明

| 交互 | 触发方式 | 响应 |
|------|----------|------|
| 切换购买/出售 | 点击购买/出售 | 切换购买/出售模式 |
| 切换分类 | 点击全部/消耗品/装备/材料 | 切换显示对应类别的商品 |
| 选择商品 | 点击商品卡片 | 选中商品，在底部详情区域显示物品信息（名称、类型、稀有度、属性/效果） |
| 购买物品 | 点击购买按钮 | 扣除金币，物品加入背包 |
| 出售物品 | 点击出售按钮 | 获得金币，物品从背包移除 |
