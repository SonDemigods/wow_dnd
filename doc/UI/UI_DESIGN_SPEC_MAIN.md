## 1. 角色创建及选择界面

### 1.1 界面概述

提供角色列表展示（最大10个角色）和角色创建功能，支持多角色管理。创建角色按钮位于角色列表最后面，当已有10个角色时不展示。点击角色卡片选中角色，底部展示进入游戏按钮，点击后使用选中角色进入主界面。角色选择界面不展示空槽位，只显示已创建的角色卡片。
